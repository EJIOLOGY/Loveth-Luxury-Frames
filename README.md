# Loveth Luxury Frames

Full-stack SAAS platform for premium luxury frames.

## 🧱 Tech Stack
- Frontend: Next.js
- Backend: Node.js + Express
- Database: MongoDB

## 📁 Structure
- client/ → Frontend app
- server/ → Backend API

## 🚀 Setup

### Client
cd client
pnpm install
pnpm dev

### Server
cd server
pnpm install
pnpm dev
